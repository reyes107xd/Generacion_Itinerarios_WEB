## npm instalados
```bash
npm install lucide-react 
npm install tailwindcss @tailwindcss/vite
npm install react-router-dom
npm install @react-oauth/google
```

## utilidad basica
Para poder visualizar la pagina realizada necesitas utilizar el siguiente comando.
```bash
npm run dev
```
